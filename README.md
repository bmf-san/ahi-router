# ahi-router
The simple URL router built with PHP

ahiru + router = ahi-router

ahiru means "duck" in japanese.

[Packagist - bmf-san/ahi-router](https://packagist.org/packages/bmf-san/ahi-router)

# Installaion
`composer require bmf-san/ahi-router`
